This for Google Analytics plugin will display Google Analytics data and statistics inside your WordPress Blog – just easy Google Analytics.

== Description ==
Using a widget, Google Analytics Dashboard displays detailed info and statistics about: number of visits within Google Analytics , number of visitors in your Google Analytics Dashboard, bounce rates, organic searches, pages per visit directly on your Google Analytics Admin Dashboard.

Authorized Google Analytics users can also view statistics like Views, UniqueViews and top searches, on frontend, at the end of each article for for Google Analytics.

Using this plugin, your Google Analytics data is collected in a fast and secure manner because Google Analytics Dashboard uses OAuth2 protocol and Google Analytics API.

Main benefits of Google Analytics:

- you can access all websites statistics in a single widget (websites within same Google Account) for Google Analytics
- cache feature, this improves loading speed up to 7 times and avoids dailyLimitExceeded, usageLimits.userRateLimitExceededUnreg, userRateLimitExceeded errors from Google Analytics API for Google Analytics
- two themes: Google Analytics Blue Theme and Light Theme for Google Analytics
- access level settings and lock profile feature for Google Analytics
- option to display Google Analytics top 24 pages, referrers and searches (sortable by columns) for Google Analytics
- option to display Google Analytics Visitors by Country on Geo Map for Google Analytics
- option to display Google Analytics Traffic Overview in Pie Charts for Google Analytics
- option to display Google Analytics statistics on frontend, at the end of each article for Google Analytics
- simple Authorization process for Google Analytics
- has multilingual support, a POT file is available for translations. If you have a complete translation, send me the translation file or upload it to our forum and will be included in next release for Google Analytics

This plugin supports Google Analytics tracking. Main Google Analytics tracking options and features:

- enable/disable google analytics tracking code for Google Analytics
- switch Google Analytics between universal analytics and classic analytics tracking methods for Google Analytics
- supports analytics.js tracking for Google Analytics comaptibility with Universal Analytics web property for Google Analytics
- supports Google Analytics ga.js tracking for comaptibility with Classic Analytics web property for Google Analytics
- track single domain, domain and all subdomains, multiple TLD domains for Google Analytics
- IP address anonymization feature for Google Analytics
- track events feature: track Google Analytics downloads, emails and outbound links (supported for both Google Analytics tracking methods: classic tracking and universal tracking) for Google Analytics

== Installation ==

1. Upload the full directory into your wp-content/plugins directory
2. Activate the plugin at the plugin administration page
3. Open the plugin configuration page, which is located under Settings -> GA Dashboard (optionally enter your API Key, Client Secret and Client ID).
4. Authorize the application using the 'Authorize Application' button
5. Go back to the plugin configuration page, which is located under Settings -> GA Dashboard to update the final settings.
